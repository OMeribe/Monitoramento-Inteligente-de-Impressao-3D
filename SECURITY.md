# Política de Segurança

## Versões Suportadas

Como este é um projeto desenvolvido no contexto do Laboratório de Informática Industrial (Labind), atualmente apenas a versão mais recente (estável) recebe atualizações e suporte.

| Versão | Suportado          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Relatando uma Vulnerabilidade

A segurança deste sistema é levada a sério, especialmente por envolver acesso a hardware de vídeo (RTSP/USB) e comunicação de rede (brokers MQTT).

Se você descobrir qualquer vulnerabilidade ou falha de segurança neste código, por favor, **não** abra uma *Issue* pública. 

Envie um e-mail com os detalhes diretamente para: **tiagoomerib@gmail.com**.

Como este é um projeto acadêmico mantido de forma independente, o tempo de resposta pode variar, mas todas as submissões serão avaliadas e respondidas o mais breve possível.
